# Welcome to simple RGB
> An RGB to hex converter

### 🏠 [Homepage](https://github.comnickylaczko/color-converter)

## Install

```sh
npm install simple-rgb
```

## Usage

```sh
const rgb = require('simple-rgb')
// or
import rgb from 'simple-rgb'

rgb(number, number, number)
```

## Run tests

```sh
npm run test
```
